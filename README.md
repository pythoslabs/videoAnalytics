# videoAnalytics
All video analytics 

How to merge videos 
Step 1 . Import moviepy.editor
Step 2 . List the files to combine 
Step 3. Use methods 'combine_clips' or 'concatenate_videoclips' to merge the vidoes
Step 4. Write the combined movie object into a file
